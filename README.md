# Robotics R&D Project

This repository contains files and information related to a robotics research and development project.

## Directory Structure Overview

*   **`data_repository/`**: Stores all raw and processed data collections, including sensory data (360 images, JSON descriptions) and environmental data (maps, floorplans).
*   **`hardware_assets/`**: Contains information about physical hardware.
    *   `robots/`: Specific details for robots like Unitree quadrupeds, drones (documentation, control interfaces, software, inventory).
    *   `components/`: Information on motors, sensors, and other electronic/mechanical parts.
    *   `equipment/`: Details about cameras, control devices, tools, and other lab equipment.
*   **`projects/`**: Houses specific sub-projects. The main initial project is `multimodal_llm_integration/` which focuses on adding advanced capabilities to robots using LLMs.
*   **`robotics_club/`**: Manages all aspects of the robotics club, including member information, events, governance, finances, courses, and meeting notes.
*   **`research/`**: Contains R&D specific research materials, such as market research, findings from surveys and interviews, and scientific literature.
*   **`logs/`**: For storing operational logs from robots, software, and experiments.

Further details about the project and the contents of each directory will be added as the project progresses.
