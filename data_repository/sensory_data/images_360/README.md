# 360-Degree Image Collection

## Overview
This directory stores 360-degree photographs captured for environmental understanding and robot localization.

## Collection Details
- **Cameras Used:** Primarily Ricoh Theta (noted as the best quality), with potentially others.
- **Approximate Count:** Around 170 images have been collected so far.
- **Methodology:** Images are typically captured from the surface of the robot, providing a robot's-eye view of the surroundings.
