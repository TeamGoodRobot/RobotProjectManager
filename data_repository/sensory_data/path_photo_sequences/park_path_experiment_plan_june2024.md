# Park Path Data Collection Experiment Plan - June 2024

## Objective
To collect sequences of images along defined paths in the park. This data will be used to test and develop robot navigation strategies, including visual path following and localization. This also serves to test strategies 'd' (Linear Photo Sequences) and 'e' (Directionality in Photos) from the `outdoor_navigation_strategies.md` document.
