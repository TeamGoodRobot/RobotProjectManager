# Multimodal LLM Integration Project

This project focuses on integrating Multimodal Large Language Models with robotic systems to enhance their capabilities.

## Outdoor Navigation Enhancement

A key focus within this project is the improvement of robot navigation capabilities in outdoor environments. Various strategies and experiments related to this are tracked in the [Outdoor Navigation Strategies document](./outdoor_navigation_strategies.md).

## Strategic Project Priorities (as of June 2024)

A primary strategic goal is the development of **"agentic" robots**. This involves creating robots that can understand complex commands, perceive their environment comprehensively, plan actions, and execute tasks with a degree of autonomy. This aligns with broader research into AI-driven interactive spaces (e.g., conceptual work on Blackboard.ai) and leverages advancements in JSON datastructures for state representation, environmental mapping (including 360 photography), and the robot's understanding of its own state and capabilities.

### Key Foundational Area: Robot Control
A critical stepping stone, currently under active development, is establishing robust **robot control mechanisms**. The current focus is on **SBUS (Serial Bus) control**, as it provides a direct and responsive way to command robot movements. This foundational control layer is essential before more complex agentic behaviors, guided by LLMs and sensory data, can be effectively implemented.

### Broader Project Options & Interests:
While agentic behavior via autonomous navigation and planning is the current key priority, other areas of interest and potential future projects include:
-   Hacking/modifying robots for ROS (Robot Operating System) and WebRTC capabilities.
-   Adding and integrating robotic arms.
-   Utilizing 360-degree cameras for comprehensive environmental data acquisition.
-   Developing practical applications like a robot serving tray.
-   Enhancing human-robot interaction by making robots appear more "cute" or relatable.
