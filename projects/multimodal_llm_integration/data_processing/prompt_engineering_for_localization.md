# Prompt Engineering for Enhanced Localization from 360° Photos

## Goal
To develop and refine prompts for use with multimodal LLMs that generate JSON descriptions from 360° photos. The focus is on producing JSON output that is optimized for robot localization in outdoor environments.

## Key Areas for Improvement in JSON Output
- **Distinguish Near vs. Far Objects:** Clearly differentiate objects based on their proximity to the robot.
- **Relative Sizing of Landmarks:** Capture information about the perceived size of distant landmarks.
- **Salient/Unique Feature Identification:** Emphasize features that are distinctive and useful for pinpointing location.
- **Consistent Structure:** Ensure a predictable and parsable JSON format.
