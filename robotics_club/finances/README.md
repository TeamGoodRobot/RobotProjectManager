# Robotics Club Finances

This section will manage information related to the club's finances, including membership fees, expenses, and budgeting.

## Initial Financial Model Outline

- **Asset Ownership:** The initial set of robots (Joule, Perl) and related equipment have been purchased by Alan (Club Founder).
- **Revenue / Cost Recovery:** The proposed model is that club membership fees will be used to contribute towards the operational costs, maintenance, and potentially future upgrades or acquisitions related to robot usage.
- **Further Details:** Specifics regarding membership fee amounts, what costs they cover, and how funds will be managed are yet to be determined.
